## 📈 Token 使用量估算

### AI 报告生成（每次）

#### 输入 Token（Input）
- **策略摘要数据**（strategy_summary）：
  - Symbol, strategy_name, spot_price: ~50 tokens
  - Legs 数据（4-6 条腿）：~300-500 tokens
  - Portfolio Greeks: ~100 tokens
  - Trade Execution: ~150 tokens
  - Strategy Metrics: ~200 tokens
  - Payoff Summary: ~150 tokens
  - **小计：~950-1,200 tokens**

- **Prompt 模板**：
  - 详细的分析提示词：~1,500-2,000 tokens
  - **总计输入：~2,450-3,200 tokens**

#### 输出 Token（Output）
- **完整分析报告**（Markdown 格式）：
  - Executive Summary: ~200 tokens
  - Market Alignment: ~400 tokens
  - Risk/Reward Analysis: ~500 tokens
  - Greeks Breakdown: ~300 tokens
  - Verdict & Score: ~200 tokens
  - **总计输出：~1,600 tokens**

#### 每次报告生成成本
```
输入成本 = 2,500 tokens × ($1.25 / 1M) = $0.003125
输出成本 = 1,600 tokens × ($5.00 / 1M) = $0.008
总成本 = $0.003125 + $0.008 = $0.011125 ≈ $0.011
```

### AI 图片生成（每次）

#### 输入 Token（Input）
- **图片生成 Prompt**：
  - 策略数据格式化：~800-1,200 tokens
  - 视觉要求描述：~500 tokens
  - **总计输入：~1,300-1,700 tokens**

#### 输出 Token（Output）
- **图片生成**：通常不按 token 计费，而是按图片数量
- **使用 Imagen 3 或类似服务**：约 $0.03 - $0.05 / 张

#### 每次图片生成成本
```
输入成本 = 1,500 tokens × ($1.25 / 1M) = $0.001875
图片生成成本 = $0.04（取中值）
总成本 = $0.001875 + $0.04 ≈ $0.042
```

## ⚠️ 风险因素

### 1. 实际使用率
- **假设**：用户每天使用满配额
- **现实**：大多数用户可能只使用 30-50% 的配额
- **影响**：实际成本可能低于估算

### 2. 其他运营成本
- **服务器成本**：AWS/Cloudflare R2 存储和带宽
- **数据库成本**：PostgreSQL 托管
- **CDN 成本**：图片和静态资源分发
- **监控和日志**：CloudWatch/类似服务
- **客服支持**：人力成本
- **营销成本**：获客成本（CAC）
- **估算**：每位 Pro 用户额外 $5-10/月

### 3. 缓存机制
- **策略 Hash 缓存**：相同策略的图片可以复用
- **影响**：实际图片生成成本可能降低 20-40%

### 4. API 价格波动
- **风险**：Google 可能调整定价
- **应对**：考虑多供应商策略（ZenMux, DeepSeek, Qwen）

## ✅ 方案可行性评估

### ✅ 优点
1. **Pro 用户定价合理**：$499/年 在保守场景下仍有 25% 净利润率
2. **普通用户成本可控**：$19.35/年，可以通过广告或升级转化覆盖
3. **缓存机制有效**：可以显著降低实际成本
4. **年付优惠**：$499/年 vs $59/月（$708/年），鼓励年付

### ⚠️ 风险
1. **高频用户风险**：如果用户每天用满配额，利润率降至 22.5%
2. **运营成本**：需要严格控制服务器、存储等成本
3. **API 价格波动**：需要监控并准备备选方案

### 💡 建议

#### 1. 定价策略优化
- **保持年费 $499**：性价比高，鼓励年付
- **月费可考虑 $69/月**：提高月付门槛，引导年付
- **普通用户**：考虑 $9.99/月 或 $99/年，覆盖成本

#### 2. 配额管理
- **Pro 用户**：保持 20/20 配额，但可以考虑：
  - 超出配额后按次收费（$0.50/次）
  - 或提供 "Unlimited" 套餐（$99/月）

#### 3. 成本优化
- **强化缓存机制**：提高缓存命中率
- **多供应商策略**：ZenMux、DeepSeek 作为备选，降低成本
- **批量处理**：优化 API 调用，减少 token 浪费

#### 4. 监控和告警
- **实时成本监控**：设置每日/每月成本告警
- **用户使用分析**：识别高频用户，考虑个性化定价
- **API 使用优化**：定期审查 token 使用量，优化 prompt
