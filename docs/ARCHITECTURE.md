# ThetaMind 架构概览

## 项目概览

**ThetaMind** 是一个 **专业级美股期权策略分析平台**，采用前后端分离的 Monorepo 架构。核心卖点是 “5 个 AI 专家 Agent + 深度研究 = 一键生成专业研报”。

---

## 技术架构

### 后端 (Python/FastAPI)

- **入口**: `backend/app/main.py` — FastAPI 应用启动
- **核心层级**:
  - `core/` (配置 & 安全)
  - `db/` (SQLAlchemy ORM & Alembic)
  - `api/` (Pydantic Schemas + 路由)
  - `services/` (AI、Tiger SDK、策略引擎、缓存、调度等)
  - `services/agents/` (多 Agent 框架)
  - `services/ai/` (AI Provider 抽象 & Gemini/ZenMux)

### 前端 (React 18 + Vite + TypeScript)

- `src/App.tsx`: Google OAuth + React Query + AuthProvider + Router
- 路由：公开页面(Landing、Login、Demo、About) & 认证页面(Dashboard、StrategyLab、Reports、Company Data、Settings) & 管理后台(Admin)
- API 层(`services/api`): Axios + JWT 拦截 + error handling

### 部署

- 本地：Docker Compose (PostgreSQL + Redis + FastAPI + React + Nginx)
- 生产：GCP Cloud Build → Cloud Run

---

## 关键业务流程示例

### 深度研报生成流程

```
User 构建策略 → 创建 Task (PENDING)
├─ Phase A: 并行/串行 5 Agents 分析 (超时 8min)
├─ Phase A+: AI 策略推荐
├─ Phase B: 深度研究 (规划 → Google Search → 综合) (超时 30min)
└─ 保存 AIReport → 扣配额 → Task SUCCESS
```
